## What is SPELLS?

SPELLS is a browser-based no-code game engine. Open it, pick a genre, build your levels, and export a fully playable game — no coding, no downloads, no accounts required.

Everything runs in your browser. Your game exports as a single HTML file you can share with anyone.

## Features

- **Full level editor** — place platforms, enemies, coins, spikes, goals and spawn points anywhere on the canvas
- **Multi-level games** — build as many levels as you want, they chain together automatically
- **6 playable characters** — Blocky, Ninja, Robot, Ghost, Cat, Ball — each with custom colors
- **Custom game screens** — design your own title screen, game over screen and victory screen
- **Background styles** — solid, stars, sky & clouds, forest, city, desert
- **Asset support** — assign any emoji or imported image to any object
- **Multi-page levels** — expand the canvas beyond one screen for larger worlds
- **Export to HTML** — one click downloads a fully self-contained playable game file
- **Works on mobile** — touch controls appear automatically on phones and tablets
- **Keyboard shortcuts in the editor** — arrows to nudge, Delete to remove, Ctrl+D to duplicate, Esc to deselect
- **High score tracking** — best score saved per game locally
- **Save / Load** — projects saved to browser localStorage

## How to use

1. Open the engine at **https://kousik2388.github.io/spells**
2. Click **New Game** and choose a genre
3. Start from a template or build from scratch
4. Use the **Level Editor** to place objects on the canvas
5. Pick your **Character** and customize colors
6. Design your **Game Screens** (title, game over, win)
7. Hit **Play** to test it
8. Click **Export** to download your game as a standalone HTML file
