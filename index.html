Tutorial

A quick guide for new creators using SPELLS without AI.

1. Choose your path

Go to Create. Pick a ready template if you want something playable fast. Pick Start from scratch if you want an empty map and full control.

2. Build the map

Use the editor tools to place walls, enemies, coins, traps and goals. Use Map pages wide to make a bigger level with Page 1, Page 2 and more.

3. Select and edit logic

Use the Select tool, click an object, then edit its behaviour in the inspector. Enemies can stay static, patrol, chase, or move in a box.

4. Rotate and resize

Select a wall/platform and change rotation to 0°, 90°, 180° or 270°. For 90°/270°, use the rotate buttons to swap width and height cleanly.

5. Add assets

Use emojis for quick prototypes, or import images. Best format: PNG/WebP/SVG, square, transparent background, around 512×512.

6. Define game rules

In Sandbox, use Game Logic / Rules to turn the empty canvas into a runner, side platformer, survival game, collect-to-win game, or balloon dodge game.

7. Build landing page

Use Game Flow to edit title, tagline, start button, second button, hero display, background style, and high score display.

8. Save & export

Press 💾 Save Project to download a .spells file — this is your editable project. To open it again, use Open Project File on the home screen. When your game is ready, press Export HTML to download the playable game file.

About SPELLS v14

SPELLS is a no-code browser game creation engine from FROM MANTRAM. It is built for beginners who want to create playable browser and mobile-ready games using templates, sandbox rules, visual editing, game-flow screens and export tools.

Version 7 focuses on product polish and clarity: the old “level editor” feeling has been replaced with a proper Build Studio, export is treated as a studio action, and landing pages are clearly part of the final game flow.
